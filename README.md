
# 👥 Employee Data Analysis Dashboard

## 📌 Project Overview

This project analyzes employee data to uncover workforce trends and business insights using Excel, Power BI, and data analysis techniques.

The dashboard helps HR teams and management understand employee demographics, salary distribution, education levels, job roles, leave balance, and workforce composition.

---

## 🎯 Objectives

- Analyze employee demographics
- Understand salary distribution across job roles
- Identify education qualification trends
- Monitor leave balance
- Explore employee age distribution
- Generate HR insights for better decision-making

---

## 🛠️ Tools Used

- Microsoft Excel
- Power BI
- Power Query
- DAX
- Git & GitHub

---

## 📊 Dashboard KPIs

- Total Employees
- Average Salary
- Average Age
- Total Leave Balance
- Male vs Female Employees
- Employees by Education Qualification
- Employees by Job Title
- Salary by Job Role
- Age Group Distribution
- Employees by Joining Year

---

## 📈 Key Insights

- Production Operators make up the largest portion of the workforce.
- Bachelor's Degree is the most common education qualification.
- Most employees are between **30–40 years** old.
- Research Scientists and Product Managers receive the highest average salaries.
- Leave balance varies significantly across departments.
- Gender distribution can be analyzed for workforce diversity.
